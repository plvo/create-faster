## Presentation

Progressive Web App support with web app manifest configuration.

[→ PWA Documentation](https://nextjs.org/docs/app/guides/progressive-web-apps)

**Supported stacks:** Next.js

## What create-faster adds

**Technical changes:**

Files added:
```
src/app/
└── manifest.ts             # Web app manifest config
```

**Manifest configuration:**
- App name and short name (uses `{appName}`)
- Icon definitions (192x192, 512x512)
- Start URL, display mode, theme colors
