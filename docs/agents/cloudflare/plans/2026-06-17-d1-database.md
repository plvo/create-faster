# Cloudflare D1 Database Option — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add Cloudflare D1 as a 4th `database` option (`d1`) in create-faster, usable identically locally and in prod (D1-everywhere), proven end-to-end on Next.js and Hono in both repo modes.

**Architecture:** D1 is special-cased so postgres/mysql/sqlite emit byte-identical output (Decision B in the spec). The `db` package gains a `createDb(binding)` factory for d1 (no `DATABASE_URL` singleton). A composable `getEnv()` (Next.js, OpenNext) is the single connection seam. The d1→cloudflare constraint is enforced **declaratively** by extending the generic `require` mechanism with a `deployment` key — NEVER a hardcoded `if (database === 'd1')`, per `.claude/CLAUDE.md` "Declarative Choice Logic".

**Tech Stack:** Bun, TypeScript, Handlebars templates, Drizzle ORM (`drizzle-orm/d1`), Wrangler (D1 migrations), Zod, `bun:test`.

**Spec:** `docs/agents/cloudflare/2026-06-17-d1-database-design.md`

## Global Constraints

- Package manager: `bun` only.
- No `any` types; no narration/temporal comments; self-documenting names (kebab-case enum values, PascalCase types, camelCase functions).
- TDD: failing test first, minimal code, green, commit. Run the relevant suite after each change; output must be pristine.
- **Declarative choice logic only** (`.claude/CLAUDE.md`): no literal branch on a choice value in core/base code; compatibility lives in META and is read generically.
- Existing databases (postgres/mysql/sqlite) MUST remain byte-identical — d1 is purely additive.
- Drizzle-only this iteration (no Prisma+D1). No auth/tRPC wiring (that is #134).
- Test harness: `apps/cli/tests/integration/helpers.ts` exposes `runCli(args: string[], cwd): Promise<{exitCode, stdout, stderr}>`, `createTempDir`, `cleanupTempDir`, `fileExists`, `readTextFile`, `readJsonFile<T>`. Rejections assert `exitCode === 1` + `stderr` contains the message.
- Run full check before final commit: `cd apps/cli && bun test` and `bun run check`.

const isProduction = process.env.NODE_ENV === 'production';

function getLocalD1DB(): string | undefined {
  const d1Dir = path.resolve(process.cwd(), '{{#if (isMono)}}../../{{/if}}.wrangler/state/v3/d1/miniflare-D1DatabaseObject');
